## Customer Events

| Event | Description |
|-------|-------------|
| `customer.created` | A new customer has been created |
| `customer.updated` | A customer has been updated |
| `customer.deleted` | A customer has been deleted |
| `customer.state_changed` | A customer's state has changed |

## Share Events

| Event | Description |
|-------|-------------|
| `share.created` | A new share has been created |
| `share.updated` | A share has been updated |

## File Sharing Events

| Event | Description |
|-------|-------------|
| `file_sharing_session.created` | A new file sharing session has been created |
| `file_sharing_session.download_completed` | A file download has been completed |
| `file_sharing_session.expired` | A file sharing session has expired |
| `file_sharing_session.payment_received` | A payment has been received for a file share |

## Workspace Events

| Event | Description |
|-------|-------------|
| `workspace.updated` | A workspace has been updated |
